# Быстрое развертывание на Vercel

## 1. Подготовка

- Убедитесь, что проект загружен на GitHub
- Создайте аккаунт на [Vercel](https://vercel.com/)

## 2. Импорт проекта

1. Перейдите на [Vercel Dashboard](https://vercel.com/dashboard)
2. Нажмите кнопку "Add New" и выберите "Project"
3. Выберите "Import Git Repository" и подключите свой GitHub аккаунт
4. Найдите и выберите репозиторий с сайтом-визиткой

## 3. Настройка проекта

- **Фреймворк**: Vite
- **Root Directory**: ./
- **Build Command**: npm run build
- **Output Directory**: dist
- **Install Command**: npm install

Обычно Vercel автоматически определяет эти настройки для Vite проекта.

## 4. Развертывание

Нажмите кнопку "Deploy" и дождитесь завершения процесса.

## 5. Доступ к сайту

После успешного развертывания вы получите URL вида:
```
https://ваш-проект.vercel.app
```

## 6. Автоматические развертывания

Vercel автоматически перестраивает и обновляет ваш сайт при каждом push в основную ветку GitHub.

---

**Примечание**: Если вы хотите использовать свой домен, вы можете настроить его в разделе "Domains" вашего проекта на Vercel.
